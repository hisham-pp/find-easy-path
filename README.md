# Shortest Path Finder (Python)

A simple Python application that finds the shortest path covering a set of points.  
You can provide **N number of coordinates**, and the program will calculate the **shortest possible path that visits all points** and report the **time taken to compute the result**.

---

## Features

- Accepts **multiple coordinate points**
- Calculates **shortest path covering all points**
- Displays **total distance**
- Measures **execution time**
- Uses **pure Python (no external dependencies)**

---

## Requirements

- Python 3.8+

No additional libraries are required.
